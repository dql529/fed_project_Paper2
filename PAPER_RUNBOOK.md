# Paper Runbook (Commands + Artifacts)

This file records the exact commands used to generate the CSV artifacts and paper figures (Fig.1-Fig.4).

All commands assume you run from the repo root (the folder that contains `r4_agg_minitest.py`).

## 0) Shell + Python notes (important)

- **cmd vs PowerShell**
  - In **cmd**, comma lists work as-is: `--seeds 0,1,2,3,4`
  - In **PowerShell**, comma lists are parsed as arrays, so **quote them**:
    - `--seeds "0,1,2,3,4"`
    - `--attack-modes "label_flip,dt_logit_scale"`
- Do **not** run PNG files via python. Open images using:
  - `start "" "paper_figs\Fig3_R4_distribution_dtD0_f5.png"`
  - or `explorer paper_figs`

## 1) Default outputs (always generated by r4_agg_minitest.py)

When you run `python r4_agg_minitest.py ...`, it writes (by default) to:

- `runs.csv`    : per-(attack, dt, f, method, seed) raw results (one row per seed)
- `summary.csv` : group-by mean/std across seeds (used for plotting error bars)
- `rounds.csv`  : per-round `W_mal` time series (weighted-only; used by Fig.2)
- `nodes.csv`   : per-node final rep/R2/R3/R4/KL (weighted-only; used by Fig.3)

You can override filenames with:
`--out-runs`, `--out-summary`, `--out-rounds`, `--out-nodes`
(If you override, also pass the same paths to `paper_figs.py` via `--summary/--rounds/--nodes`.)

## 2) Fig.1 - Clean macro-F1 (holdout) vs f (3 attacks, 3 subplots)

### 2.1 Run the sweep (D0 example)

cmd:
```bat
cd /d <repo_root>
python r4_agg_minitest.py --attack-modes label_flip,stealth_amp,dt_logit_scale --dt-levels D0 --mal-nodes 0,3,5 --methods weighted,mean,median,trimmed_mean --seeds 0,1,2,3,4
```

### 2.2 Generate the figure

```bat
python paper_figs.py --figs 1 --dt D0
```

Outputs:
- `paper_figs/Fig1_cleanF1_vs_f_dtD0.png`

## 3) Fig.2 - W_mal vs round (focus f=5; at least label_flip + dt_logit_scale)

### 3.1 Run the minimal sweep for Fig.2 (weighted only)

python r4_agg_minitest.py --attack-modes label_flip,dt_logit_scale --dt-levels D0 --mal-nodes 5 --methods weighted --seeds 0,1,2,3,4

## 4) Fig.3 - R4/KL distribution (benign vs malicious)

This figure uses `nodes.csv`. In our pipeline, `nodes.csv` logs one row per client at the **final round** for each seed.
So if `N=10`, `f=5`, `seeds=5`, you should see `n_benign=25`, `n_malicious=25`.

### 4.1 Run the minimal sweep for Fig.3

```bat
cd /d <repo_root>
python r4_agg_minitest.py --attack-modes dt_logit_scale --dt-levels D0 --mal-nodes 5 --methods weighted --seeds 0,1,2,3,4
```

### 4.2 Generate the figure

```bat
python paper_figs.py --figs 3 --dt D0 --f-fig3 5
```

Outputs:
- `paper_figs/Fig3_R4_distribution_dtD0_f5.png`
- `paper_figs/Fig3_R4_distribution_dtD0_f5_caption.txt` (caption w/ statistical unit + pass-rate)

## 5) Fig.4 - Ablation (full / R4-only / R2-only / no-gate)

We generate a dedicated ablation summary CSV, then plot from it.

### 5.1 Run ablation sweeps (label_flip + dt_logit_scale)

```bat
cd /d <repo_root>
python r4_agg_minitest.py --attack-modes label_flip,dt_logit_scale --dt-levels D0 --mal-nodes 0,3,5 --methods weighted_full,weighted_r4only,weighted_r2only,weighted_nogate --seeds 0,1,2,3,4 --out-runs ablation_runs.csv --out-summary ablation_summary.csv
```

### 5.2 Generate Fig.4 from ablation_summary.csv

```bat
python paper_figs.py --figs 4 --dt D0 --ablation-summary ablation_summary.csv --ablation-attacks label_flip,dt_logit_scale
```

Outputs:
- `paper_figs/Fig4_ablation_dtD0.png`

## 6) Multi-DT (D0/D1/D2) in one run

If you want to compute everything in one sweep across DT fidelities:

```bat
cd /d <repo_root>
python r4_agg_minitest.py --attack-modes label_flip,stealth_amp,dt_logit_scale --dt-levels D0,D1,D2 --mal-nodes 0,3,5 --methods weighted,mean,median,trimmed_mean --seeds 0,1,2,3,4
```

Then generate per-DT figures by selecting dt:

```bat
python paper_figs.py --figs 1 --dt D0
python paper_figs.py --figs 1 --dt D1
python paper_figs.py --figs 1 --dt D2
```
